**Tofu Zen** is a theme meticulously crafted to provide a focused, clean, and distraction-free coding experience. Inspired by the softness of tofu and the tranquility of Zen philosophy, this theme utilizes a balanced color palette that is high in contrast yet gentle on the eyes for long-duration sessions.

## ✨ Key Features
- 🌿 **Zen Focus:** A curated color palette designed to reduce eye strain.
- 🎨 **Minimalist:** A clean UI without unnecessary visual distractions.
- 🛠️ **Zed Optimized:** Built specifically to leverage the performance and aesthetic strengths of Zed IDE.

## 🚀 Installation

### Via Zed Extension Manager
1. Open **Zed**.
2. Open the **Command Palette** (`Ctrl+Shift+P` on Linux/Windows or `Cmd+Shift+P` on macOS).
3. Type `zed: extensions` and press **Enter**.
4. Search for **Tofu Zen**.
5. Click **Install**.

## ⚙️ Activation
Once installed, follow these steps to activate the theme:

1. Open the **Command Palette**.
2. Type `theme selector: toggle` (or use the shortcut `Ctrl+K Ctrl+T` / `Cmd+K Cmd+T`).
3. Select **Tofu Zen Dark**.

## 🔧 Recommended Settings
For the best experience, add the following to your `settings.json`:

```json
{
  "theme": "Tofu Zen Dark",
  "ui_font_size": 16,
  "buffer_font_family": "JetBrains Mono",
  "buffer_font_size": 14
}
```
